Add yearly folders here for the public Key Moments page.

Example:
- `public/key-moments/2023/`
- `public/key-moments/2024/`
- `public/key-moments/2025/`

Supported image formats:
- `.jpg`
- `.jpeg`
- `.png`
- `.webp`
- `.avif`
- `.heic`

The site now keeps each year intentionally curated and shows up to 4 images per year.
Public titles/captions use generic polished labels by default instead of raw file names.
For exact titles/captions, use overrides in `src/lib/localKeyMoments.ts`.
